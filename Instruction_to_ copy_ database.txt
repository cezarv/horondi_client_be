1. Install MongoDatabase tools  https://www.mongodb.com/try/download/database-tools
2. Open cmd 
3. Move to directory with installed tools to folder bin
4. Execute "mongodump.exe --uri="mongodb+srv://user-read-write1:<password>@horondi-yl27g.mongodb.net/<dbname>?retryWrites=true&w=majority" "
 and set valid credentials
5. Copy of database will be created at dump folder
6. Then, if you want to restore database locally:
7. Execute "mongorestore.exe dump/<dump_name> --db="<name_restored_db>" "
